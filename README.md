# Matt Davies - Data Analysis Protfolio

# About
Hi, I’m Matt. I have a background in Computer Science and currently work as a secondary school teacher in Wales, where I lead Personal and Social Education (PSE) and Relationships and Sexuality Education (RSE).

Through my role, I regularly work with data to monitor progress, identify trends, and support decision-making. I have applied data analysis skills to explore School Health Research Network (SHRN) data, generating insights which I have presented to senior leadership and governors to inform school-wide strategies.

I am currently transitioning into data analysis, with practical experience in Python (Pandas, Matplotlib), SQL, and data visualisation tools such as Power BI and Tableau.

I enjoy working with data to clean, analyse, and present insights in a clear and meaningful way for different audiences.

This repository showcases my data analysis projects, demonstrates my technical skills, and documents my ongoing development as a data analyst.
